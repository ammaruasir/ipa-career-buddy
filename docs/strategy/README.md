## الجمهور المستهدف للمنصة

- **الخريجون** المستعدّون للوظائف الحكومية
- **الموظفون الحكوميون الحاليون** الساعون للترقّي
- **طلاب IPA** المسجّلون في برامج المعهد

## مبادئ التصميم الجوهرية

1. **التدريب أولاً، التقييم استثناء** — الافتراضي = ممارسة خاصة بالطالب
2. **تعليم الإطار، لا الدرجة فقط** — STAR/CAR قبل الأرقام
3. **اعرض، لا تقل** — نسخة نموذجية بجانب نسخة الطالب
4. **عربية أولاً، إنجليزية ثانياً** — RTL كامل، Noto Naskh Arabic، ثنائية اللغة
5. **حماية بيانات صارمة** — PDPL، 90 يوماً للممارسة، تشفير عند الراحة

## ترتيب التنفيذ المعتمَد

```
P0.1 (وضع التدريب)
    ↓
P0.2 (تغذية راجعة لكل إجابة)
    ↓
P0.6 (طبقة المدرّس والدفعات)
    ↓
P0.5 (بنك أسئلة IPA)
    ↓
P0.3 (تقييم السيرة الذاتية)
    ↓
P0.4 (منشئ السيرة الذاتية)
```

**التبرير:** المنتج بحاجة "هيكل التعلّم" قبل "أدوات التعلّم". الفصل بين التدريب والتقييم + طبقة المدرّس + التغذية الراجعة هي الـ"عظام"؛ السيرة الذاتية وبنك الأسئلة هي الـ"عضلات".
