

## خطة تحسين الإرشاد المهني ووضع التدريب

### الوضع الحالي
- **الإرشاد المهني**: صفحة بسيطة بزر واحد "ابدأ التحليل" → تعرض نتائج ثابتة (مسارات، مهارات، وظائف مطابقة) بدون تفاعل إضافي
- **وضع التدريب**: 3 أزرار فقط (نصي/صوتي/فيديو) بدون أي تتبع للتقدم أو توصيات تدريبية

### التحسينات المقترحة

#### 1. تحسين الإرشاد المهني
- **إضافة توصيات دورات تدريبية**: تعديل prompt الذكاء الاصطناعي ليقترح دورات/شهادات مناسبة لكل مهارة مطلوبة (مثل CCNA، ICDL، PMP) مع روابط لمنصات مثل Coursera/Udemy
- **تحليل فجوة المهارات بصرياً**: رسم بياني رادار يقارن مهارات المرشح الحالية بمتطلبات الوظائف المطابقة
- **خطة تطوير شخصية**: خريطة طريق زمنية (3-6-12 شهر) بخطوات واضحة
- **حفظ النتائج**: تخزين نتائج التحليل في قاعدة البيانات لتتبع التطور عبر الوقت

#### 2. تحسين وضع التدريب
- **لوحة تقدم التدريب**: عرض عدد التدريبات المكتملة ومتوسط الدرجات لكل نوع مقابلة
- **توصيات ذكية**: بناءً على نتائج المقابلات السابقة، اقتراح نوع المقابلة والمجال الذي يحتاج تحسين
- **أهداف تدريبية**: مثل "أكمل 3 مقابلات صوتية هذا الأسبوع" مع شريط تقدم
- **نصائح قبل التدريب**: عرض نصائح مخصصة بناءً على نقاط الضعف من المقابلات السابقة

### التنفيذ التقني

#### الملفات المعدّلة:

| الملف | التعديل |
|-------|---------|
| `supabase/functions/career-guidance/index.ts` | تحديث الـ prompt لإضافة `training_plan` و `recommended_courses` للنتائج |
| `src/pages/CareerGuidance.tsx` | إضافة أقسام: الدورات المقترحة، خطة التطوير الزمنية، رسم الرادار |
| `src/pages/CandidateDashboard.tsx` | تحسين قسم التدريب: إضافة إحصائيات التدريب، توصيات ذكية، أهداف |

#### تعديل الـ AI Prompt (career-guidance):
إضافة حقلين جديدين للنتائج:
```json
{
  "training_plan": [
    {"phase": "الشهر 1-3", "goals": ["..."], "actions": ["..."]}
  ],
  "recommended_courses": [
    {"name": "...", "platform": "Coursera/Udemy", "skill": "...", "duration": "..."}
  ]
}
```

#### تحسين لوحة التدريب (CandidateDashboard):
- جلب المقابلات التي `practice=true` وحساب الإحصائيات
- عرض بطاقات بـ: عدد التدريبات، متوسط الأداء، آخر تدريب
- اقتراح المجال الأضعف للتدريب عليه بناءً على أقل درجة تقييم
