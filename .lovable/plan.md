

## المشكلتان

### 1. تكرار الحروف في الصوت
عندما يرد الذكاء الاصطناعي بنص فيه حروف مكررة مثل "مررررررررررررة" أو "جميل مررررررة"، يتم إرسال النص كما هو إلى TTS فيقرأ كل حرف مكرر. الحل: تنظيف النص قبل إرساله لـ TTS بإزالة التكرارات الزائدة.

### 2. كثرة الأسئلة التتبعية
حالياً البرومبت يسمح بسؤال تتبعي واحد لكل سؤال رئيسي، لكن عملياً الذكاء الاصطناعي يتجاوز هذا. الحل: تعديل البرومبت لتحديد أقصى **3 أسئلة تتبعية** قبل الانتقال الإجباري، وإضافة عداد follow-up في الكود.

## التعديلات

### 1. `src/hooks/useLiveInterview.ts` — تنظيف النص قبل TTS
إضافة دالة `cleanTextForTTS` تزيل الحروف المكررة أكثر من مرتين:
```typescript
const cleanTextForTTS = (text: string): string => {
  // "مررررررررة" → "مرة"  |  "جميييييل" → "جميل"
  return text.replace(/(.)\1{2,}/g, '$1');
};
```
واستخدامها في `speakText` قبل إرسال النص.

### 2. `supabase/functions/chat/index.ts` — تعديل البرومبت + عداد
- تعديل قاعدة الأسئلة التتبعية: أقصى **3 أسئلة توضيحية** لكل سؤال رئيسي، وبعدها انتقل إجبارياً للسؤال التالي.
- إضافة تعليمة صريحة: "إذا وصلت لـ 3 أسئلة تتبعية متتالية، ابدأ السؤال التالي بـ [NEW_Q] حتى لو الإجابة غير مكتملة."
- إضافة تنظيف النص في الرد أيضاً (إزالة الحروف المكررة من رد الذكاء الاصطناعي نفسه).

### الملفات المعدّلة
