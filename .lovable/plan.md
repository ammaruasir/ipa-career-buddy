

## التحليل

لوحة الإدارة الحالية (`AdminDashboard.tsx`) تعرض **المقابلات** فقط مع زر "عرض" — لا يوجد قسم لإدارة المستخدمين (عرض، تعديل دور، حذف).

## الحل: إضافة قسم إدارة المستخدمين

### التعديلات

| الملف | التعديل |
|-------|---------|
| `src/pages/AdminDashboard.tsx` | إضافة قسم "إدارة المستخدمين" يعرض جدول بجميع المستخدمين مع أزرار التحكم |

### المميزات المضافة

1. **جدول المستخدمين** — يعرض: الاسم، البريد، الدور الحالي، تاريخ التسجيل
2. **تغيير الدور** — قائمة منسدلة لتبديل الدور (مرشح / HR / أدمن) مباشرة من الجدول
3. **حذف مستخدم** — زر حذف مع تأكيد (AlertDialog) قبل الحذف
4. **إضافة مستخدم** — ليست ممكنة مباشرة من الواجهة لأن التسجيل يتم عبر نظام المصادقة، لكن يمكن إضافة زر دعوة مستخدم جديد عبر رابط التسجيل
5. **بحث وتصفية** — فلتر بالاسم والدور

### ملاحظة أمنية
- حذف المستخدم يتم عبر edge function تستخدم `service_role_key` لحذف المستخدم من `auth.users` (لا يمكن حذفه من الكلاينت مباشرة)
- تغيير الدور يتم عبر جدول `user_roles` الموجود حالياً مع سياسة RLS تسمح للأدمن بالإدارة

### الملفات الجديدة

| الملف | الغرض |
|-------|-------|
| `supabase/functions/manage-user/index.ts` | Edge function لحذف المستخدمين (تحتاج `service_role_key`) |
